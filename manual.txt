서버 여는 법과 닫는 법

1. 터미널 열기

PowerShell 또는 명령 프롬프트를 엽니다.

2. 프로젝트 폴더로 이동하기

아래 명령어를 입력합니다.

cd "D:\학습공동체"

3. 서버 열기

아래 명령어를 입력합니다.

python -m http.server 8000

정상적으로 실행되면 터미널에 서버가 실행 중이라는 문구가 표시됩니다.
브라우저에서 아래 주소로 접속합니다.

http://localhost:8000

특정 파일을 바로 보고 싶으면 아래처럼 접속할 수 있습니다.

http://localhost:8000/index.html

4. 서버 닫기

서버를 실행한 터미널 창에서 아래 키를 누릅니다.

Ctrl + C

종료할지 물어보면 Y를 입력하고 Enter를 누릅니다.

5. 같은 와이파이에 연결된 다른 기기에서 접속하기

같은 와이파이에 연결된 휴대폰, 태블릿, 다른 컴퓨터에서는 localhost 주소를 사용할 수 없습니다.
localhost는 서버를 연 PC 자기 자신을 뜻합니다.

다른 기기에서는 서버를 연 PC의 IP 주소로 접속해야 합니다.
현재 이 PC의 접속 주소는 아래와 같습니다.

http://192.168.0.2:8000

예를 들어 서버를 열어 둔 상태에서 휴대폰 브라우저에 위 주소를 입력하면 접속할 수 있습니다.

만약 접속이 안 되면 아래 내용을 확인합니다.

- 서버를 연 PC와 접속하려는 기기가 같은 와이파이에 연결되어 있는지 확인합니다.
- Windows 방화벽에서 Python 또는 해당 서버 프로그램의 개인 네트워크 접근을 허용합니다.
- 서버를 열 때 사용한 포트 번호가 8000번인지 확인합니다.

6. 내 IP 주소 다시 확인하기

IP 주소는 네트워크 환경에 따라 바뀔 수 있습니다.
다시 확인하려면 터미널에서 아래 명령어를 입력합니다.

ipconfig

그 다음 인터넷에 연결된 어댑터의 IPv4 주소를 확인합니다.
현재는 아래 주소를 사용합니다.

192.168.0.2

7. 8000번 포트가 이미 사용 중일 때

아래처럼 다른 포트 번호로 실행합니다.

python -m http.server 8080

그 다음 브라우저에서 아래 주소로 접속합니다.

http://localhost:8080

같은 와이파이의 다른 기기에서는 아래 주소로 접속합니다.

http://192.168.0.2:8080

8. Python 명령어가 안 될 때

아래 명령어를 대신 사용해 봅니다.

py -m http.server 8000

그래도 실행되지 않으면 Python이 설치되어 있지 않은 상태일 수 있습니다.
